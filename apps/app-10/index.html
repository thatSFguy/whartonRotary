Location Access Needed

This app needs your location to calculate distances to each hole. It is not stored or shared — it stays on your phone. Here's how to allow it:

  1. First, check Location Services are ON:
    Settings → Privacy & Security → Location Services → make sure the toggle at the top is green
  2. Then allow Safari to use location:
    While still in Location Services, scroll down and tap Safari Websites → select While Using the App
  3. Allow this website:
    Go back to Settings → scroll down → tap Safari → scroll to Settings for Websites → tap Location → set to Ask
  4. Come back to this page and tap Try Again below. Safari will ask permission again — this time tap Allow
